# eggnog - a cryptocurrency ticker inside terminal 
Eggnog is python based crypto ticker. It will work inside a terminal. You have to open the 'eggnog.py' within the terminal with <code>python3 eggnog.py</code> line.

For now, you don't have an option, as its only a web-scraper with bitcoin currency scraper, and not much code.

Bitcoin price is scraped from [coinlivewatch.com](https://www.livecoinwatch.com/).
